# arc-db README

## Features

- Add Database Connection
- Refresh Connections
- Execute SQL Query
- View Table Data
- Edit Connection
- Delete Connection
- New Query
- Export Data

## Requirements

- Node.js
- npm
- TypeScript
- Visual Studio Code

## Extension Settings

This extension contributes the following settings:

* `arc-db.connections`: Array of database connection configurations.
